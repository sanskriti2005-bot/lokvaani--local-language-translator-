# lokvaani--local-language-translator-
Lok Vaani is a handy app that breaks down language barriers in India by blending real-time translation with fun learning, travel tips, and cultural insights all in one spot. It's perfect for tourists navigating new regions, students picking up local dialects, or anyone curious about India's incredible diversity.
It's perfect for tourists navigating new regions, students picking up local dialects, or anyone curious about India's incredible diversity. people who type or speak text in one Indian language (like Hindi, English, Tamil, or more), and it instantly translates to others such as Marathi, Kannada, Bengali, and beyond. Supports speech-to-text (talk and it types) and text-to-speech (hears the translation aloud). Powered by APIs like Google Translate for accuracy, with offline backups for common phrases in languages like Assamese, Gujarati, Nepali, Bhojpuri, and Haryanvi.
